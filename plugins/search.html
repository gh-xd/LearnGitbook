搜索插件

搜索插件

gitbook-plugin-searh-pro

介绍

Gitbook也自带搜索功能，搜索内容没问题，但是没有强调色。

安装

在npm项目文件夹下执行

npm install gitbook-plugin-search-pro

只支持Gitbook >= 3.0

安装完毕后，可以在package.json中找到已安装的依赖包

同时，项目文件夹下也会出现一个名为node_modules的文件夹

配置

根据文档描述

安装完后，在book.js中添加插件的配置

{
    plugins: ['search-pro'];
}

效果

搜索结果可以高亮

菜单折叠插件

菜单折叠插件

gitbook-plugin-expandable-chapters

介绍

折叠左侧导航栏菜单。

安装

在npm项目文件夹下执行

npm gitbook-plugin-expandable-chapters

配置

在book.js中添加

{
    plugins: ["expandable-chapters"];
}

效果

折叠一级菜单
